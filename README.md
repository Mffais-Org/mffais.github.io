#### To build the project: 

1. Install the node_modules: `yarn install` or `npm install`
2. Build: `app yarn` build or `npm run build` 
3. Deploy "build" folder

#### To run project

1. Install the node_modules: `yarn install` or `npm install`
2. Run project: `yarn start` or `npm start`
